ドラッグ=移動 / ホイール=ズーム / ホバー=その日の暦を表示

免責事項 — 本システムは過去データと暦の相関を検証するための研究・教育用ツールです。 将来の値動きを予測・保証するものではなく、投資助言でもありません。 為替相場を確実に予測できるシステムは存在しません。売買の判断と結果はすべて利用者自身の責任です。

このシステムの考え方

「暦と相場に相関がある」という仮説を前提にせず、過去約27年分の日次データ(欧州中央銀行の参照レート)に対して 暦のカテゴリごとの平均リターン・勝率を計算し、全期間平均との差が統計的に意味を持つか(z値・p値)を数字で確かめます。 相関が「あった」も「なかった」も等しく検証結果です。

  • チャート — 値動きに暦とイベントを重ねて目視確認。一目の時間論による変化日候補も投影。
  • 暦検証 — 曜日 / 五十日 / 月相 / 二十四節気 / 二十七宿 / 日干五行(火水) / 数秘 / 水星逆行 / 五十音暦 / 一目基本数値日 の統計表。
  • 類似局面 — 現在のチャート形状と相関の高い過去の局面を検索し、その後の値動きと当時の社会現象・暦を提示。
  • 年表 — 主要な社会現象の当日の暦と相場反応を横断表示。

暦計算の定義(流派差に注意)

  • 二十四節気 — 太陽黄経15°刻み(天文計算・JST基準)。国立天文台の暦要項と日単位で一致することを2026年分で確認済み。
  • 七十二候 — 太陽黄経5°刻み(本朝七十二候・伊勢神宮略本暦準拠の配当)。
  • 二十七宿(宿曜) — 2026旧暦年(2026/2/17〜2027/2/5)は旧暦法: 各旧暦月の朔日に宿を配当し月内で27循環(出典: 火水の会基準暦典 / sukuyou.divination.page。 原資料の注記どおり旧3〜12月の朔日宿は「要確認」扱い)。 それ以外の期間は月の黄経からアヤナムシャ(既定24°)を引いて13°20′で区分する天文学的定法で代用し、 表示に(旧暦法)/(天文)と方式を明記。方式が異なる期間が混在する点は統計の解釈時に注意。
  • 六曜 — 旧暦月+旧暦日から算出。旧暦テーブルを持つ2026旧暦年のみ表示・集計(それ以外はデータなし)。
  • 火水の法則 — 日の十干の五行で判定(火=丙・丁の日、水=壬・癸の日)。日干支は1949-10-01=甲子を基準とした厳密計算。
  • 言霊五十音暦(五十連) — 冬至点起点方式(火水の会基準暦典/TK_ANCHOR_03): 冬至点の瞬間を第1音ホの起点とし、1音=7日7時間12分(365日÷50音)で「言霊一言法則」の記載順 (ホに起こりマに終わる五十連)を進める。第50音マは次の冬至点までを受け持つ。 日への割当はその日の正午(JST)時点の音。冬至点は太陽黄経270°通過を天文計算(基準暦典の値と±数分で一致、 2026年の全50期間で基準表との一致を確認済み)。この規則で1999年〜全期間に遡って適用。 各音の定義は「言霊法則」タブに原文のまま収録。
  • 水星逆行 — ケプラー軌道要素からの計算。留の日は実際と±1日程度ずれることがあります。
  • 一目均衡表の時間論 — 直近スイング(前後5本の極値)を1日目として基本数値 9,17,26,33,42,52,65,76 を投影。

統計を読むときの注意

  • 多数のカテゴリを同時に検定しているため、偶然だけでも |z|≥2 は数個現れます(多重検定の問題)。
  • 本物のアノマリーかを見分ける目安: ①複数の通貨ペアで同方向に出る ②期間を前半/後半に分けても再現する ③理屈(実需フロー等)が説明できる。
  • ECBレートは1日1本の参照値のため、日中の値動きや高値・安値は反映されません。

設定

データソース

為替: Frankfurter API(欧州中央銀行の日次参照レート、1999年〜)。 ブラウザから直接取得し、当日分は端末内(localStorage)にキャッシュします。サーバーは使いません。

暦の基準: 「2026年暦データベース(火水の会 基準暦典)」— 二十四節気・朔弦望は国立天文台 令和8年暦要項、 五十音暦は TK_ANCHOR_03(冬至点起点)、宿曜は sukuyou.divination.page(旧暦法)、 七十二候は伊勢神宮略本暦準拠。言霊の定義は「言霊一言法則」写本テキスト(原文のまま収録)。
